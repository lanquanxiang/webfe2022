# 2. CSS的编写和使用
1. 编写规则 selector { property1:value1; property2:value2;……}
2. 使用方法（四种）
	1. 链接式：在外部编写css文件，link链接进网页，href设置CSS的URL
	2. 导入式：在外部编写css文件，style标签中，@import
	3. 内部式：在style标签中编写
	4. 行内式：在HTML的标签中编写，样式写在style属性中
