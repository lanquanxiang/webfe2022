# 1. CSS定位
## 1. position
1. static	默认		不脱离文档流		元素不能移动
2. relative	相对		不脱离文档流		可以移动，相对与原来的位置
3. absolute	绝对		脱离文档流		可以移动，相对于具有特殊定位的父元素
4. fixed	固定		脱离文档流		可以移动，相对与窗口
注意：元素的显示和隐藏不影响文档布局，元素会叠加在其他元素之上，考虑脱离文档流
## 2. 偏移量
