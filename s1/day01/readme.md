# 2. HTML概述
1. HTML的作用：内容+结构
2. HTML的概念：开始标记、结束标记、属性、文本
3. HTML的特点
	1. 解释执行
	2. 不可见字符会被解析为一个空白符
	3. 不区分大小写，建议小写
	4. 建议标签正确嵌套
4. 网页的构成：声明、结构标记（html(root)、head、body）
