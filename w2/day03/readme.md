## 2. 组合选择器
1. 后代 E F
2. 父子 E>F

5. 群组选择器 E,F
## 3. 属性选择器
1. 存在attr属性		[attr]
2. 存在attr属性，且值等于val  [attr=val]
3. 存在attr属性，且值以a开头  [attr^=a]
4. 存在attr属性，且值以b开头  [attr$=b]
5. 存在attr属性，且值包含c    [attr*=c]
